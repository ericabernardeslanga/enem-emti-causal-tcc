# Escopo do Projeto

## Tema

Analise causal aplicada a dados do ENEM e do Ensino Medio em Tempo Integral.

## Motivacao

Investigar relacoes entre politicas educacionais e desempenho, permanencia ou indicadores associados a trajetorias escolares, com foco em resultados observaveis em bases publicas.

## Entrega academica

- trabalho escrito final;
- base analitica documentada;
- scripts reproduziveis;
- graficos e tabelas para defesa e apresentacao.

## Entrega profissional

- repositorio bem estruturado no GitHub;
- narrativa de projeto orientada a ciencia de dados aplicada;
- pipeline reutilizavel para outros estudos de politicas publicas.

## Decisoes a fechar

- recorte temporal;
- unidade de analise;
- conjunto final de bases integradas;
- estrategia causal principal;
- variaveis de desfecho e controles.
