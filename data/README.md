# Dados

Esta pasta centraliza os dados do projeto.

- `raw/`: arquivos originais baixados das fontes publicas.
- `external/`: insumos de terceiros ja tratados externamente.
- `interim/`: bases intermediarias geradas ao longo da limpeza.
- `processed/`: base analitica pronta para modelagem.

Regras:

- nao editar manualmente arquivos em `raw/`;
- documentar fonte, data de download e filtros aplicados;
- manter nomes de arquivos consistentes e descritivos.
